# Linux system check

Prosty skrypt do sprawdzenia: Docker, Docker Compose, Python, Java, uzycie dysku i CPU oraz IntelliJ/PyCharm.

## Uruchomienie

```bash
python3 run_check.py
```

JSON:

```bash
python3 run_check.py --json
```

## Uwagi

- Skrypt jest przygotowany pod Linux; na innych systemach wyniki moga byc niepelne.
- Nie wymaga dodatkowych zaleznosci.
